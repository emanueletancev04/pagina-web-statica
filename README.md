# Landing Page per Corso di Web Development

Questo repository contiene il codice sorgente per una landing page statica e completamente responsive, creata come esercizio pratico per dimostrare le competenze di base in HTML5 e CSS3.

Il progetto simula la pagina di presentazione di un corso di sviluppo web, con un layout moderno a due colonne che si adatta a diverse dimensioni dello schermo.


---

##  Caratteristiche Principali

*   **Struttura Semantica HTML5:** Utilizzo di tag moderni come `<header>`, `<nav>`, `<main>`, `<aside>` e `<footer>` per una migliore accessibilità e SEO.
*   **Layout Moderno con Flexbox:** Il layout principale a due colonne (contenuto e sidebar) è gestito interamente con CSS Flexbox, una delle tecniche fondamentali per l'impaginazione web.
*   **Design Responsive (Mobile-First):** La pagina è progettata per essere visualizzata correttamente su tutti i dispositivi. Le Media Query vengono utilizzate per adattare il layout, trasformandolo a colonna singola su schermi più piccoli.
*   **Header Sticky:** L'intestazione della pagina, comprensiva di hero section e barra di navigazione, rimane fissata in alto durante lo scorrimento per una migliore esperienza utente.
*   **Stile Pulito e Manutenibile:** Il CSS è organizzato e commentato, con l'utilizzo di variabili CSS (`:root`) per una gestione centralizzata dei colori e dei font, rendendo il codice più facile da leggere e modificare.

##  Scopo del Progetto

Questo progetto è stato creato con i seguenti obiettivi di apprendimento:

1.  **Consolidare le basi di HTML e CSS.**
2.  **Mettere in pratica la creazione di layout complessi e responsive.**
3.  **Dimostrare la capacità di tradurre un design visivo in codice funzionante.**
4.  **Creare un progetto "vetrina" pulito e professionale da aggiungere al proprio portfolio.**
